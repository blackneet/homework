Экспресс-файлы — сервис обмена информацией

Вы можете анонимно воспользоваться сервисом хранения, пересылки и обмена файлов, в
этом случае доступ к объекту будет возможен только с помощью ключа, который Вы получите
при создании объекта.

Экспресс-письмо

Если получатель не зарегистрирован в системе, сообщение всё равно будет доставлено.
На указанный номер телефона будет выслана SMS с информацией о доступе к аккаунту.
